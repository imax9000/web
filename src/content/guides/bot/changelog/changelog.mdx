### 0.3.3
- Support for video posts.
- Added the <Reference to="/docs/bot/classes/Bot#getProfiles" type="Method" /> method.

### 0.3.2

- Minor further bug fixes to event handling.
- Fixed a bug resulting in an error when attempting to create or negate labels.

### 0.3.1

- The `like` event will now be emitted for likes on posts, feed generators, and labeler profiles.
- Fixed the `repost` and `follow` events not including the `uri` property.

## 0.3.0

- Added support for Starter Packs with the <Reference to="/docs/bot/classes/StarterPack" /> class.
- New properties/methods on <Reference to="/docs/bot/classes/Post" />: `embeddingDisabled`, `quoteCount`, `getQuoteCount()`, `getQuotes()`.
- Removed the <Reference to="/docs/bot/classes/Bot#declareLabeler" type="Method" /> and <Reference to="/docs/bot/classes/Bot#deleteLabelerDeclaration" type="Method" /> methods. Use the [@skyware/labeler](https://skyware.js.org/docs/labeler) package instead.
- Fixed rate limit being stricter than it should've been ([#3](https://github.com/skyware-js/bot/issues/3)).
- Fixed a bug where the number of event emitters would multiply.

### 0.2.1

- <Reference to="/docs/bot/types/PostPayload#facets" type="Property" /> can now contain <Reference to="/docs/bot/classes/Facet" /> instances.

## 0.2.0

- **NEW:** Chat!
  - Initialize your bot with `emitChatEvents: true` and listen for chat messages with `bot.on("message", (message) => {})`. Read more about it in [Chatting with Users](/guides/bot/introduction/chatting-with-users)!
  - The <Reference to="/docs/bot/classes/Conversation" /> class represents a conversation with a user.
  - The <Reference to="/docs/bot/classes/ChatMessage" /> and <Reference to="/docs/bot/classes/DeletedChatMessage" /> classes represent chat messages.
  - New properties/methods on <Reference to="/docs/bot/classes/Profile" />: `incomingChatPreference`, `getConversation()`, `getMessages()`, `sendMessage()`.
  - New methods on <Reference to="/docs/bot/classes/Bot" />: `getConversationForMembers()`, `getConversation()`, `listConversations()`, `getConversationMessages()`, `sendMessage()`, `sendMessages()`, `leaveConversation()`, `setChatPreference()`.
- **NEW:** Labeling!
  - You can now subscribe to and publish labels. Note that you will need an active [Ozone instance](https://github.com/bluesky-social/ozone/blob/main/HOSTING.md) to be able to publish labels.
  - The <Reference to="/docs/bot/classes/Labeler" /> class represents a labeler service.
  - New properties/methods on <Reference to="/docs/bot/classes/Profile" />: `isLabeler`, `getLabeler()`, `labelAccount()`, `negateAccountLabels()`, `labelProfile()`, `negateProfileLabels()`.
  - New methods on <Reference to="/docs/bot/classes/PostReference" /> and <Reference to="/docs/bot/classes/Post" />: `label()`, `negateLabels()`.
  - New methods on <Reference to="/docs/bot/classes/List" />: `label()`, `negateLabels()`.
  - New methods on <Reference to="/docs/bot/classes/FeedGenerator" />: `label()`, `negateLabels()`.
  - New methods on <Reference to="/docs/bot/classes/Bot" />: `getLabeler()`, `getLabelers()`, `addLabeler()`, `removeLabeler()`, `label()`, `negateLabels()`, `declareLabeler()`, `deleteLabelerDeclaration()`.
- Now uses file headers to determine content type when an image URL is used in a post.
- Embeds now correctly link to the specified URL.
- `@` will be automatically prepended to mentions created with the <Reference to="/docs/bot/classes/RichText" /> class.
- Added the <Reference to="/docs/bot/classes/Bot#putRecord" type="Method" /> method.
- Several redundant interfaces for `Bot` methods have been replaced with <Reference to="/docs/bot/types/BaseBotGetMethodOptions" />.
